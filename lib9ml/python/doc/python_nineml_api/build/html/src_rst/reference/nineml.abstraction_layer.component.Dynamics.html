nineml.abstraction_layer.component.Dynamics¶

class nineml.abstraction_layer.component.Dynamics(regimes=None, aliases=None, state_variables=None)¶

A container class, which encapsulates a component’s regimes, transitions, and state variables

__init__(regimes=None, aliases=None, state_variables=None)¶

Dynamics object constructor

Parameters:
  • aliases – A list of aliases, which must be either Alias objects or ``string``s.
  • regimes – A list containing at least one |Regime| object.
  • state_variables – An optional list of the state variables, which can either be StateVariable objects or string s. If provided, it must match the inferred state-variables from the regimes; if it is not provided it will be inferred automatically.
