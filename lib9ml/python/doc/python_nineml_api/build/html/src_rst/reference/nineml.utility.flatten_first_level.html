nineml.utility.flatten_first_level¶

static utility.flatten_first_level(nested_list)¶

Flattens the first level of an iterable, i.e.

>>> flatten_first_level( [ ['This','is'],['a','short'],['phrase'] ] ) 
['This', 'is', 'a', 'short', 'phrase'] 

>>> flatten_first_level( [ [1,2],[3,4,5],[6] ] ) 
[1,2,3,4,5,6]
