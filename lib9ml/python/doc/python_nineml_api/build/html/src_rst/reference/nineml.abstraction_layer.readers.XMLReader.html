classmethod read(filename, component_name=None)¶

Reads a single ComponentClass object from a filename.

Parameters:
  • filename – The name of the file.
  • component_name – If the file contains more than one ComponentClass definition, this parameter must be provided as a string specifying which component to return, otherwise a NineMLRuntimeException will be raised.
Return type:

Returns a ComponentClass object.
